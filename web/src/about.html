Sobre o ytmdown

Inspiração

Meu nome é Guilherme, gosto muito de música, mas algo que dificulta muito é você usar os downloaders de música do YouTube e vim com os dados da musica todos bagunçados. Então para manter minha biblioteca sempre organizada eu criei o ytmdown, ele permite o download de música do YouTube com todas as tags(nome, artista,foto do album) já embutidos. Para isso ele faz uma busca no Deezer por dados da música.

Open-Source(Código Aberto)

Este projeto é um projeto de código aberto disponível no GitHub
